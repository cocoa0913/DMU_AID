개인정보처리방침 (Privacy Policy) - 디뮤 에이드 (DMU AID)
최종 수정일: 2026년 3월 21일

1. 개요
디뮤 에이드(DMU AID, 이하 "본 확장 프로그램")는 동양미래대학교 학생들의 개인정보 보호를 최우선으로 생각합니다. 본 방침은 본 확장 프로그램이 사용자의 정보를 어떻게 처리하는지 설명합니다. 본 프로그램은 학생들의 e-Class(LMS) 활동을 효율적으로 관리할 수 있도록 돕는 로컬 기반의 학습 보조 도구로 설계되었습니다.

2. 수집 및 이용하는 개인정보 항목
본 확장 프로그램은 핵심 기능을 제공하기 위해 아래와 같은 정보를 브라우저 내부에서 처리합니다.

인증 정보 (Authentication Info): 로그인 기능을 제공하기 위해 사용자의 학번(아이디)과 비밀번호를 처리합니다. 이 정보는 동양미래대학교 공식 e-Class 서버(eclass.dongyang.ac.kr)와 보안 세션을 수립하기 위한 목적으로만 사용됩니다.

웹사이트 콘텐츠 (Website Content): 사용자의 e-Class 계정으로부터 과제 마감 기한, 공지사항, 알림 내역 및 강좌 정보를 직접 가져와 파싱합니다.

사용자 생성 콘텐츠 (User Content): 사용자가 직접 입력한 To-Do 리스트 항목을 처리합니다.

3. 개인정보의 저장 및 보안
로컬 저장 원칙: 수집된 모든 학업 데이터와 To-Do 항목은 chrome.storage.local을 사용하여 사용자의 로컬 브라우저 환경에만 저장됩니다.

외부 전송 불가: 본 확장 프로그램은 어떠한 사용자 데이터, 인증 정보, 학업 정보도 개발자의 서버나 제3자의 외부 서버로 전송하지 않습니다. 모든 데이터 처리는 사용자의 기기 내에서만 이루어집니다.

암호화 통신: 모든 인증 및 데이터 수집 과정은 학교 공식 서버와의 HTTPS 보안 프로토콜을 통해 안전하게 진행됩니다.

4. 개인정보의 보유 및 파기
사용자가 브라우저에서 본 확장 프로그램을 삭제하면, 로컬에 캐시된 모든 학업 데이터와 To-Do 리스트는 즉시 파기됩니다.

본 프로그램은 기능 수행을 위해 필요한 로컬 세션 외에 어떠한 사용자 활동 기록도 별도로 유지하지 않습니다.

5. 제3자 제공 및 공유
본 프로그램은 사용자의 정보를 외부에 판매, 거래 또는 공유하지 않습니다. 본 프로그램에는 구글 애널리틱스 등의 제3자 추적 스크립트나 광고 스크립트가 일절 포함되어 있지 않습니다.

6. 크롬 웹스토어 정책 준수
디뮤 에이드는 구글 크롬 웹스토어 사용자 데이터 정책(Limited Use 포함)을 준수합니다. 본 프로그램은 기능 구현에 필요한 최소한의 권한(storage, alarms, sidePanel, host_permissions)만을 요청합니다.

Privacy Policy for DMU ADE (디뮤 에이드)
Last Updated: March 21, 2026

1. Introduction
DMU ADE ("the Extension") is committed to protecting the privacy of students at Dongyang Mirae University. This Privacy Policy explains how the Extension handles information. The Extension is designed as a local-first productivity tool to help students manage their e-Class (LMS) activities efficiently.

2. Information Collection and Usage
The Extension processes the following types of data solely to provide its core features:

Authentication Information: To provide the login feature, the Extension handles the user’s student ID and password. This information is used only to establish a secure session with the official e-Class server (eclass.dongyang.ac.kr).

Website Content: The Extension fetches and parses academic data—including assignment deadlines, notices, and course notifications—directly from the user's e-Class account.

User-Generated Content: The Extension stores the To-Do list items manually entered by the user.

3. Data Storage and Security
Local Storage Only: All retrieved academic data and personal To-Do items are stored exclusively in the user's local browser environment using chrome.storage.local.

No External Transmission: The Extension does not transmit any user data, authentication credentials, or academic information to any external servers or third parties. All data processing occurs locally within the user's browser.

Encryption: Authentication is handled through the university's official secure protocols via HTTPS.

4. Data Retention
All locally cached data and To-Do items are immediately deleted when the user uninstalls the Extension from their browser.

The Extension does not maintain any persistent records of user activity beyond the local session required for functional use.

5. Third-Party Disclosure
We do not sell, trade, or otherwise transfer your information to outside parties. The Extension does not include any third-party tracking, analytics (e.g., Google Analytics), or advertising scripts.

6. Compliance with Chrome Web Store Policies
DMU ADE complies with the Google Chrome Web Store User Data Policy, including the requirements for Limited Use. The Extension only requests the minimum permissions necessary to function: storage, alarms, sidePanel, and host permissions for eclass.dongyang.ac.kr.
